# MindTalk 优化清单

## 一、内容创作增强

- [x] **草稿箱** — 发帖时可存草稿，个人中心查看/编辑未发布草稿。难度：中
- [x] **@提及用户** — 帖子/评论中输入 `@` 触发用户搜索，选中后生成通知。难度：中
- [x] **Markdown 编辑器升级** — 支持表格、流程图(Mermaid)、代码沙箱嵌入。难度：中
- [x] **帖子系列/合集** — 作者可将多篇帖子归入同一系列，详情页展示上一篇/下一篇。难度：中
- [x] **置顶/加精时效** — 置顶和加精支持过期时间，到期自动取消。难度：低

## 二、社交互动

- [x] **用户勋章/成就系统** — 根据发帖数、获赞数、连续登录等发放勋章。难度：中
- [x] **评论按热度排序** — 评论支持点赞数排序。难度：低
- [x] **分享功能** — 生成帖子分享卡片（OG 元标签），复制链接/生成海报。难度：低
- [x] **阅读历史** — 记录用户最近浏览的帖子，个人中心可查看和清除。难度：低
- [x] **阅读列表** — 类似「稍后阅读」，与收藏区分开的独立列表。难度：低

## 三、个性化 Feed

- [x] **关注动态流** — 独立「关注」Tab，只展示已关注作者的新帖子。难度：中
- [x] **关注标签/分类** — 用户可订阅标签/分类，首页推荐相关帖子。难度：中
- [x] **个性化推荐** — 基于阅读历史和互动行为的协同过滤推荐。难度：高

## 四、通知与消息

- [x] **通知设置** — 用户可配置哪些类型通知需要推送。难度：低
- [x] **私信聊天** — 用户间 1v1 私信（`conversations` + `messages` 表已建好）。难度：中
- [x] **消息已读/未读** — 通知列表区分已读/未读状态（`is_read` 字段已有）。难度：低
- [x] **邮件通知** — 关注作者发新帖/收到回复时发送邮件。难度：中

## 五、内容发现

- [x] **周榜/月榜** — 按时间段统计最热帖子排行榜。难度：低
- [x] **相似帖子推荐** — 基于 ES `more_like_this` 推荐内容相似帖子。难度：中
- [x] **RSS 订阅** — 提供分类/标签/用户的 RSS Feed。难度：低
- [x] **全文搜索增强** — 支持按用户/分类/时间范围筛选，搜索结果高亮。难度：中

## 六、管理后台增强

- [x] **操作日志查询** — `admin_logs` 表已建好，补充分页查询和前端展示。难度：低
- [x] **敏感词过滤** — 发帖/评论时自动检测替换敏感词，可配置词库。难度：中
- [x] **内容批量操作** — 批量删除帖子/评论、批量封禁用户。难度：中
- [x] **数据看板图表** — 统计页增加图表（日活、留存率、热门时段）。难度：中

## 七、用户体验

- [x] **快捷键支持** — 发帖 `Ctrl+N`、返回首页等全局快捷键。难度：低
- [x] **无限滚动** — 首页帖子列表自动加载下一页（IntersectionObserver）。难度：低
- [x] **回到顶部** — 长页面右下角悬浮按钮。难度：低
- [x] **PWA 支持** — Service Worker + manifest，离线访问和桌面安装。难度：中

## 八、性能与基础设施

- [x] **CDN 图片处理** — 上传图片自动压缩/裁剪/转 WebP。难度：中
- [x] **接口限流完善** — `@RateLimit` 已有，补全局限流配置和前端 429 提示。难度：低
- [x] **评论虚拟滚动** — 评论列表支持虚拟滚动优化长列表性能。难度：中

---

## 优先级

| 优先级 | 功能 |
|--------|------|
| P0 | @提及用户 + 关注动态流 + 私信聊天 + 通知已读/设置 + 草稿箱 |
| P1 | 敏感词过滤 + 评论热度排序 + 阅读历史 + 勋章系统 + 帖子系列 |
| P2 | 个性化推荐 + PWA + 邮件通知 + RSS 订阅 + 数据看板图表 |
